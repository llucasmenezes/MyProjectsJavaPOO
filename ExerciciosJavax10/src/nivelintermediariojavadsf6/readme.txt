Desenvolver um sistema em Java para gerenciar uma lista de ninjas usando a estrutura de dados LinkedList. Cada ninja terá nome, idade e vila. inicialmente sete ninjas serão adicionados à lista, e o sistema deverá permitir operações como remover o primeiro ninja, adicionar um novo ninja no inicio e exibir a lista completa. também será possível acessar um ninja em uma posição específica da lista.
